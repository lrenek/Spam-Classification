
INSTALLATION INSTRUCTIONS

For the program to function effectively, python must be installed.

Python Installation:
1. From the official Python website (https://www.python.org/) select and download the appropriate file, according to the specifications of your computer.
2. Run the installer. Make sure to check the boxes that say “Install Launcher for All Users” and “Add Python to PATH”.

Pip Installation:

1. Open the command prompt
2. Find the path where Python and the other libraries we have installed are stored
3. Download the get-pip.py file with the curl command https://bootstrap.pypa.io/get-pip.py -o get-pip.py
4. Type the command: python get-pip.py

Install Necessary Libraries: 

Numpy:
To install, type the command at the command prompt: pip install –upgrade numpy.
Pandas:
To install, type the command at the command prompt: pip install –upgrade pandas.
Scikit-learn:
To install, type the command at the command prompt: pip install –upgrade scikit-learn.
